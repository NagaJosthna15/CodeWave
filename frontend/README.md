# CodeWave Frontend

This is the frontend of the CodeWave coding platform.

## Features

- User Authentication
- Overview Dashboard
- Coding Problems
- Profile Management
- React Router Navigation

## Tech Stack

- React.js
- CSS
- React Router
- Axios / Fetch API

## Run Locally

```bash
npm install
npm start
```

## Deployment

Frontend: Vercel

Backend: Render
